# Zera Manga Api

## Installation

- Install dependencies
  ```bash
  > npm i
  ```
- Build server
  ```bash
  > npm run build
  ```
- Run server
  ```bash
  > npm start
  ```
- Server listening on `http://127.0.0.1:4000`

## Endpoint

- /popular
- /hot
- /latest
- /colored
- /search/:query

- /manga/page/:pageNumber
- /manhua/page/:pageNumber
- /manhwa/page/:pageNumber

- /detail/:detailSlug
- /chapter/:detailSlug

- /genre
- /genre/:genreSlug/page/:pageNumber
